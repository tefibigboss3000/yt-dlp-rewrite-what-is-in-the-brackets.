# yt-dlp-rewrite-what-is-in-the-brackets.
When downloading music using yt-dlp, the files come with a string of characters inside brackets—for example, "A Stranger [Qypuo3T4aHQ]." The program automatically replaces these characters after you have downloaded all your songs, eliminating the need to rename them one by one, for now, only works for mp3, wav and flac files


You can run it in cmd, if you want, you can open it in visual studio and replace the variables in the script however you like.


obviously, you will need Python installed on your PC to run it via the command prompt (cmd). save the file in a folder, type "cmd" in the address bar, and then run it using the command `python script_name.py`.
